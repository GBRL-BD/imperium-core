Acerca de Imperium Core

Imperium Core es un bot multifunción diseñado para ofrecer herramientas de economía, moderación, bienvenida y entretenimiento para comunidades de Discord.


Nuestro objetivo es proporcionar una experiencia estable, intuitiva y en constante evolución para administradores y usuarios, reuniendo múltiples sistemas en una sola solución.

Características

💰 Economía

Sistema de monedas, recompensas, rankings y progresión.

🛡️ Moderación

Herramientas para mantener el orden y la seguridad.

👋 Bienvenida

Mensajes automáticos y configuraciones personalizadas.

🎮 Entretenimiento

Funciones para aumentar la interacción de la comunidad.

Términos de Servicio

Última actualización: 9 de junio de 2026


  • Cumplir los términos y políticas de Discord.
  • No abusar de las funciones del bot.
  • No explotar errores o vulnerabilidades.
  • No utilizar el bot para actividades ilegales.
  • El acceso podrá restringirse por incumplimiento de estas normas.

Política de Privacidad

Información almacenada:

  • IDs de usuarios.
  • IDs de servidores.
  • Configuraciones personalizadas.
  • Datos económicos y estadísticas.
  • Información necesaria para la moderación.

Los datos se utilizan exclusivamente para el funcionamiento de Imperium Core y no son vendidos ni compartidos con terceros, salvo obligación legal.


Las solicitudes de eliminación de datos pueden realizarse mediante: soporteimperiumcorebot@gmail.com
